# Lavavel 9

Projet de site d'un restaurant construit avec Laravel 9.
## Prérequis
 vscode
- `composer`
- `artisan`
- `npm`
## Installation

```bash
composer install
npm install
```


## Utilisation

Dans un premier terminal, lancez :
Php artisan serve

Dans un autre terminal, lancez :
npm run dev

Pour voire la page d'accueil, ouvrire : [ Accueil](http://127.0.0.1:8000)
